Wat is 2FA?

Tweefactorauthenticatie voegt een extra beveiligingslaag toe aan je account. Naast je wachtwoord heb je een code nodig die elke 30 seconden verandert.

Hoe werkt het?

  1. Installeer een authenticator app (Google Authenticator, Authy, of Microsoft Authenticator)
  2. Scan de QR code met de app
  3. Voer de code in om te verifiëren
